The libraries provide MTP interface to onboard microSD card in a Teensy 3.5/3.6 board via a connected host PC.

For **Teensyduino version 1.39**, please do this to test MTP
1.  Clone SdFat libraries from https://github.com/greiman/SdFat into your sketchbook libraries folder or Teensy3 library folder
2.  Test the example program from the library.

See the history in https://forum.pjrc.com/threads/43050-MTP-Responder-Contribution?p=138772
